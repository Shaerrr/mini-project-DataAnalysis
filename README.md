# DLC 수익성 분석 및 개발방향 도출 프로젝트
## 프로젝트 기획

프로젝트 배경
1) 게임의 수명 연장
2) 사용자 요구 반영 및 소통
3) 유저의 게임 이탈 방지
4) 적은 투자 비용 및 새로운 수익 창출

프로젝트 목적
- 다양한 DLC 출시를 한 심즈4로 타겟으로
  심즈4 DLC 성공 요인 파악
- 시뮬레이션 게임의 DLC 개발 방향성 제시

사용할 데이터
- 스팀 리뷰(리뷰 개수, 긍적/부정 평가, 리뷰 텍스트 분석)
- 스팀DB에서 플레이어 수 분석
- 트위치에서 스트리밍 시청자 수 분석

기대 적용 방안
- 크래프톤 사에서 심즈4와 비슷한 인생 시뮬레이션 장르를 25년 상반기에 출시 예정인데
  우리가 분석한 자료를 토대로 방향성 제시
